## Description

This project is my CS50 Final Project. It is a mobile stock management application developed using Flutter and SQLite.

The application allows users to manage product inventories by adding, editing, deleting, and searching products. Stock levels are tracked locally on the device, and products that reach critical stock levels are visually highlighted. Additionally, all product data can be exported as a PDF document.

The main goal of this project is to provide a simple, offline-capable inventory tracking solution while demonstrating practical use of Flutter, local databases, and file generation.

---

## Features

- Add new products with name, category, quantity, minimum stock level, and price
- Edit existing product information
- Delete products with confirmation
- Real-time product search
- Visual indication for low-stock (critical) products
- Export all products as a PDF file
- Offline local data storage using SQLite

---

## Technologies Used

- Flutter (Dart)
- SQLite (sqflite)
- PDF generation (pdf, printing)
- Material Design

## How to Run the Project

1. Install Flutter:
   https://flutter.dev/docs/get-started/install

2. Clone the repository:
git clone https://github.com/muhammedozd/cs50-final-project-stock-management.git

3. Navigate to the project directory:
cd cs50-final-project-stock-management

4. Install dependencies:
   flutter pub get
   
5. Run the application
   flutter run
